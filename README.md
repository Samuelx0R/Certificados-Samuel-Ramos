# 📜 Repositório de Certificados - Samuel Ramos

Este repositório foi criado para organizar e centralizar os certificados obtidos em cursos, bootcamps, trilhas de aprendizagem e projetos concluídos ao longo da minha formação acadêmica e profissional.

Os arquivos estão organizados por **instituição/plataforma** e, quando aplicável, por **trilha ou bootcamp**, com o objetivo de facilitar a navegação, a consulta e a atualização contínua deste acervo.

## 📂 Estrutura do Repositório

A organização segue a estrutura abaixo:

```bash
certificados/
├── instituicao/
│   ├── trilha-ou-bootcamp/
│   │   ├── certificado-1.pdf
│   │   ├── certificado-2.pdf
│   │   └── certificado-final.pdf
│   └── curso-avulso.pdf
```

## 🎯 Objetivo

Manter um histórico organizado das certificações conquistadas ao longo da minha jornada de aprendizado, reunindo cursos concluídos em diferentes instituições e plataformas.

Este repositório será atualizado continuamente com novas certificações.
